ClimateStrike AI is an open source app for the purpose of the early prediction of natural disasters.

Install Open source Try it online -->

Our Mission

Natural disasters are a large problem in the global community. We strive to provide services for the early prediction of natural disasters. We use 14 different artificial intellgience models that take in user input of weather predictions to forecast natural disasters with high accuracy. The models only require 5 numerical inputs iwth the capability of predicting a natural disaster. These models are available for free and the software with them is licensed under the Apache 2.0 License

Accurate Predictions

95%+ Accuracy on most predictions, with few false negatives. Predictions are valid for any times with the same conditions

Built for Preformance

Predictions in less than 3 seconds on CPU with little RAM overhead. When installation is not an option, use the free demo to preserve system resources

Stability

Installers and other binaries provided for MacOS and Windows 10/11 as well as source code for universal builds


Open source

All of our code and models are open source and available for the public to download, view, fork and use. They are hosted on GitHub and Huggingface.


Model files are available at the ClimateStrike-Core repository and on Huggingface
