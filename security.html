Security Policy

Last updated: 11 October 2025 09:55 AM EDT

Data Collection

ClimateStrike AI is designed with privacy in mind. We do not collect, store, or transmit any personally identifiable information (PII) or sensitive user data. The app operates locally and does not require user accounts, cloud syncing, or background data collection.

Open Source Transparency

Our codebase is fully open source and available on GitHub. This ensures transparency and allows the community to audit, contribute to, and verify the security of the application.

Third-Party Services

ClimateStrike AI does not integrate with third-party analytics, advertising, or tracking services EXCEPT Huggingface Spaces and GitHub. We do our best not transmit user data.

Please also read the security and privacy policies of Huggingface and Github for a complete view.

Security Practices

  • All code is reviewed and tested before release.
  • Dependencies are regularly updated to patch known vulnerabilities.
  • We encourage responsible disclosure of any security issues via GitHub Issues.

Contact

If you have questions or concerns about security, please email me at sharktide@bugcrowdninja.com.

We reserve the right to update and change this security / privacy policy at any time.
